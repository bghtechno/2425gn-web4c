Apa Itu HTML ?

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman website dan aplikasi web4. Bukan bahasa pemrograman, HTML digunakan untuk menyusun bagian-bagian seperti paragraf, heading, dan tautan (link) pada halaman web4. HTML menggunakan tag untuk menandai elemen-elemen konten, yang memberitahu browser web cara menampilkan konten6.
